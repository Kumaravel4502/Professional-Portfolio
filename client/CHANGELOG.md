# Changelog

## v1.2 (Latest)
- **Tech Stack Upgrade**: Migrated entire codebase to React + TypeScript (Vite).
- **Design System**: Implemented a premium dark theme with glassmorphism using Tailwind CSS 4.
- **Resume Integration**: Fully updated Hero, About, Experience, Skills, Certifications, and Education sections with mandatory resume data.
- **Animations**: Integrated Framer Motion for scroll reveals, hover effects, and smooth transitions.
- **Projects & Contact**: Redesigned existing sections to match the premium aesthetic while preserving functionality.
- **SEO & Accessibility**: Added semantic HTML tags, meta descriptions, and Google Fonts (Inter & Outfit).
- **Clean Structure**: Standardized folder structure and removed legacy `.jsx` files.
