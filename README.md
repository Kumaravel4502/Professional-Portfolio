# 🚀 Stunning MERN Portfolio (JavaScript Edition)

Beautiful, client-attracting portfolio with MongoDB visitor counter + Nodemailer contact form.

1. `cp .env.example .env` and fill your details
2. MongoDB Atlas connection string + Gmail App Password
3. Run:

```bash
# Terminal 1 - Backend
cd server
npm install
npm run dev

# Terminal 2 - Frontend
cd ../client
npm install
